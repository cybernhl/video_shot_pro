# video_shot_pro

#### 项目介绍
隐云图解制作-更方便的视频截图与台词拼接应用

#### APP说明
已编译版本详见：[隐云图解制作](https://www.coolapk.com/apk/153839)

使用本APP可以实现快速对视频截图并且拼接成长图，可用于制作电影图解，台词截图等
亦可以用于截取GIF动图，也支持对其他视频播放器播放的视频进行截图并拼接。
更多功能详见各大应用市场。
